KRYONIX KNOWLEDGE BASE

Metode Pembayaran

Kami menerima berbagai metode pembayaran yang aman, mulai dari Transfer Bank (BCA, Mandiri), E-Wallet (Dana, OVO, Gopay), hingga QRIS. Semua transaksi diproses otomatis.

Lama Server Aktif

Servers kami menggunakan sistem instan deploy. Layanan aktif dalam hitungan detik setelah pembayaran dikonfirmasi oleh sistem.

Garansi Uang Kembali

Kami memberikan garansi 7 hari jika layanan tidak dapat digunakan karena faktor teknis dari pihak kami. Kesalahan pengguna tidak kami cover.

Upgrade Spesifikasi

Bisa. Kami mengizinkan kustomisasi resource (CPU/RAM) tanpa perlu install ulang. Silakan hubungi support melalui WhatsApp Admin.

Lokasi Server

Server kami tersebar di Singapore (SG), Jakarta (ID), dan Amerika (US) untuk latensi rendah.

Server Down?

Kami menjamin uptime 99.9%. Jika terjadi maintenance terjadwal, kami akan memberitahukan melalui email dan status dashboard.
